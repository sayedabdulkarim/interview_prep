## Interview Mein Kaise Answer Karna Hai

```
Step 1: CLARIFY - "Kya ye production mein hai? Kitne users hain?"
Step 2: IDENTIFY - Root cause kya ho sakta hai (2-3 possibilities)
Step 3: DIAGNOSE - Kaise confirm karoge (tools, logs, metrics)
Step 4: FIX - Short-term fix + Long-term solution
Step 5: PREVENT - Aage se kaise rokoge (monitoring, tests, CI/CD)
```

Ye framework har scenario question pe use karo.
