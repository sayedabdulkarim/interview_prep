What is the React Native Bridge?

    JS code (React Native):

    "JavaScript/TypeScript code jo UI aur app logic handle karta hai."

    Native code:

    "Java/Kotlin (Android) aur Swift/Obj-C (iOS) jo hardware-level features access karta hai."


    So, bridge is like communication layer between Js and  native code ...


What is the new architecture (Fabric + TurboModules) in RN?

React Native’s new architecture consists of Fabric and TurboModules, built on top of JSI.
Fabric is the new rendering system that gives faster and synchronous UI updates.
TurboModules make native modules lazy-loaded & faster.
JSI removes the old bridge, allowing JS and native to talk directly.
Together, the app becomes faster, smoother, and more memory efficient.

What is the role of JSI (JavaScript Interface)?
    - JSI removes the old bridge, allowing JS and native to talk directly.
